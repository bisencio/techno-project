Vision & Mission

Vision Statement

"To build a future where every Filipino can avoid long queues for essential services, positioning the barangay as the model of efficient and people-centered local governance."

Mission Statement

"Barangay Connect Scheduler aims to use popular digital tools like Messenger and SMS to provide an easy-to-use, transparent, and accessible appointment system that cuts down waiting time."

The Problem We Solve

Long Physical Queues

Residents waste hours waiting in line for basic services

Inefficient Manual System

No visibility on wait times or queue status

Limited Accessibility

Not everyone has smartphones or internet

Target Users

Working Professionals

Need quick service during breaks

Senior Citizens

Prefer SMS, need accessibility

Homemakers

Busy schedules, need flexibility

Students

Tech-savvy, prefer Messenger

Technical Requirements

Low-cost, scalable infrastructure designed for local government units

Database

  • PostgreSQL/MySQL
  • Users, Appointments, Queue
  • 5GB initial storage

Hosting

  • DigitalOcean ($5-12/mo)
  • Free tier: Railway/Render
  • 1GB RAM, 1 vCPU min

SMS Gateway

  • Semaphore (₱0.35-0.50)
  • Chikka API alternative
  • Two-way SMS support

Messenger Bot

  • Facebook Graph API (Free)
  • Dialogflow/Wit.ai NLP
  • Quick Replies & Buttons

Backend

  • Node.js/Python Flask
  • REST API endpoints
  • Webhooks for updates

Admin Dashboard

  • React/Vue web app
  • Queue management
  • Role-based access
